# Lee Charles Portfolio

## Overview

This is the source code for my personal portfolio website. It showcases my projects, skills, and professional experience as a Software Developer and Data Analyst.

## Features

- Responsive design for optimal viewing on all devices
- Dark mode theme for a modern, sleek look
- Project showcase with detailed descriptions
- Skills section highlighting my technical expertise
- Smooth navigation and scrolling effects

## Tech Stack

- React.js
- Chakra UI for styling and components
- React Router for navigation
- GitHub Pages for hosting

## Installation

To run this project locally:

1. Clone the repository:
   ```
   git clone https://github.com/lee-cha-dev/portfolio.git
   ```
2. Navigate to the project directory:
   ```
   cd portfolio
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm start
   ```

## Usage

After starting the development server, open [http://localhost:3000](http://localhost:3000) to view it in the browser. The page will reload if you make edits.

## Deployment

This site is deployed using GitHub Pages. To deploy your own version:

1. Update the `homepage` field in `package.json` with your GitHub Pages URL.
2. Run `npm run deploy` to build and deploy the site.
